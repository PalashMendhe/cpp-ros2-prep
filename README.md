# C++ ROS2 Prep

14-day C++ fundamentals study before ROS 2 Lyrical installation.

## Structure
- day01/ - Variables, types, functions, compilation
- day02/ - Conditionals, loops, header files
- day03/ - Compilation, CMake basics
- day04/ - Pointers and references
- day05/ - Memory management, smart pointers
- day06/ - Stack vs heap, unique_ptr, shared_ptr
- day07/ - Classes and constructors
- day08/ - Inheritance and virtual functions
- day09/ - const correctness, this pointer
- day10/ - STL vectors and maps
- day11/ - STL queues, arrays, iterators
- day12/ - Lambdas, auto, range-based for
- day13/ - std::bind, modern C++17
- day14/ - CMake, multi-file projects
